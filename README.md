## Use

Download
```
git clone git@github.com:avysel/tezos-scratch-dapp.git
cd tezos-scratch-dapp
```

Install
```
npm install
```
Build for local
``` 
npm run build
```

Run on local server
```
npm run start
```

Build for deploy
```
npm run predeploy
```

Deploy on Github Pages
```
npm run deploy
```
